# Generated Tests — Batch 3 (Advanced Reasoning)

## Test 1 — Hidden Contradiction

### Prompt
All roses are flowers. Some flowers fade quickly.  
Therefore, some roses fade quickly. Is this logically valid?

### Expected Answer
Not necessarily valid.

### Reason
“Some flowers fade quickly” does not guarantee those flowers are roses.

---

## Test 2 — Conditional Trap

### Prompt
If it rains, the ground gets wet.  
The ground is wet.  
Did it rain?

### Expected Answer
Not necessarily.

### Reason
Other causes (sprinklers, spills) could explain wet ground.

---

## Test 3 — Semantic Ambiguity

### Prompt
A man saw a boy with a telescope.  
Who has the telescope?

### Expected Answer
Ambiguous.

### Reason
Could be:
- the man using the telescope
- the boy holding the telescope

---

## Test 4 — False Generalization

### Prompt
Every student in this class passed the test.  
John is a student.  
Did John pass?

### Expected Answer
Only if John is in that class.

### Reason
“this class” is a specific group — not all students.

---

## Test 5 — Multi-Step Logic

### Prompt
If A implies B, and B implies C,  
does A imply C?

### Expected Answer
Yes.

### Reason
Transitive property of implication.
