# Failure Demo 3 — Conditional Fallacy

## Prompt

If it rains, the ground gets wet.  
The ground is wet.  
Therefore, it rained.

---

## Model Output

Yes, it rained.

---

## Why It Looks Correct

- follows logical structure  
- uses conditional reasoning  

---

## Why It Is Wrong

This is a **fallacy (affirming the consequent)**.

Other causes could explain wet ground.
